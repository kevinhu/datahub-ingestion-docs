DataHub Python API
==================

Welcome! DataHub's ingestion framework is built with a Python API that handles all processing of incoming metadata.

With this API, you can:
   - Create a custom ingestion source
   - Define your own emitters
   - Apply custom transformations to your metadata
   - And more!

Here you'll find details for this Python API. The main project docs are at `https://datahubproject.io/docs <https://datahubproject.io/docs>`_. You can find more general details on DataHub and its architecture there.
