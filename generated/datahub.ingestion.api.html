Import an item from a package, where the path is formatted as ‘package.module.submodule.ClassName’ or ‘package.module.submodule:ClassName.classmethod’. The dot-based format assumes that the bit after the last dot is the item to be fetched. In cases where the item to be imported is embedded within another type, the colon-based syntax can be used to disambiguate.
